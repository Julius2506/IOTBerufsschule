**Beschreibung des Projekts**:
Ziel dieses Projekts ist die Entwicklung eines IoT-basierten Überwachungssystems für ein Terrarium. Das System erfasst die Umweltbedingungen kontinuierlich und stellt diese übersichtlich für den Benutzer dar.
Funktionsweise & Technische Details
• Datenerfassung: Verschiedene Sensoren sind an einen Mikrocontroller (Arduino) angeschlossen, um folgende Parameter zu messen:
• Temperatur
• Luftfeuchtigkeit
• Lichtintensität
• Verarbeitung: Die erfassten Messwerte werden ausgelesen, verarbeitet und an eine Benutzeroberfläche übertragen.
• Visualisierung: Auf einem intuitiven Dashboard (GUI) werden alle relevanten Daten in Echtzeit angezeigt, sodass der Zustand des Terrariums jederzeit überwacht werden kann.
